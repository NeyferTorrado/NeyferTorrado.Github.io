Inicio de Sesion, Sistema de Ventas

Nuestra interfaz inicio de sesion cuenta con la opcion de recuperar tu contraseña por medio del correo electronico, solo debes tener tu informacion totalmente actualizada, tambien verifica el ingreso a sesion con usuario y contraseña registrados en la base de datos del sistema, con esto queremos brindarte mayor seguridad y confiabilidad al momento de proteger tu informacion

Interfaz de venta, Sistema de Ventas

Nuestra interfaz de ventas tiene el objetivo de ser facil y rapida al momento de utilizarce, esta cuenta con la funcionalidad para descontar los productos del inventario, tambien cuenta con la generacion del numero de factura de forma automatica este se imprime junto con la factura.

Ingreso de mercancia, Sistema de Ventas

Nuestra interfaz para el Ingreso de mercancia tiene el objetivo de ser facil y rapida al momento de utilizarce, esta cuenta con la funcionalidad para sumar los productos al inventario, tambien cuenta con la generacion del numero de factura de forma automatica.

Interfaz Registro de empleados, Sistema de Ventas

Por medio de esta interfaz vas a poder gestionar los accesos al sistema a tus empleados, y llevar el registro de las personas que ingresan al sistema, tambien podras gestionar los permisos que le daras a los accesos creados

Registro de Productos, Sistema de Ventas

Por medio de esta interfaz podras crear tus produtos con las caracteristicas que desees y el sistema te arrojara el precio de venta al momento de guardar la informacion, podras gestionar el codigo de tus productos a tu preferencia

Interfaz de reportes, Sistema de Ventas

En esta interfaz podras ver a detalle tus ventas, tus compras y el ingreso de mercancia que has tenido durante el tiempo gestionado, te facilitara la opcion de busqueda para que filtres tu informacion por años, meses o dias segun tu preferencia.
